# MMFE-Based-Stochastic-Replenishment-Dataset
A synthetic multi-period, multi-scenario benchmark dataset designed for the AP replenishment optimization problem. The dataset is generated using rolling MMFE-style demand forecasts and conditional scenario sampling. The current benchmark instance contains 15 products, 80 materials, 8 decision periods, 8 scenarios per period, and a 3-period rolling forecast horizon, making it suitable for evaluating stochastic replenishment, inventory control, and MAGA/AP optimization algorithms.
